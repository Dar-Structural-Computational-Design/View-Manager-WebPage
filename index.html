0100

Revit Sheet

Sheet production is the most repetitive work in a Revit project. These three modules turn it into selection plus one click.

0101 · REVIT SHEET

Create Revit Sheet

Pick the views, pick the title block, pick the viewport type — View Manager creates a sheet per view and places the viewport for you. Works with parent and dependent views alike.

  1. Choose Parent view or Dependent view as the source list.
  2. Search and tick the views to sheet. Multi-select is the point.
  3. Pick a Title block type from the project's loaded families.
  4. Pick a Viewport type (e.g. NoTitle).
  5. Hit Create Sheet. One sheet per selected view, viewport placed.
0102 · REVIT SHEET

Add Sheet Revision

Issuing a revision across a 200-sheet set shouldn't mean opening 200 sheets. Select the sheets, select the revision, apply once.

  1. Search and select the target sheets.
  2. Choose the revision to stamp.
  3. Apply — the revision lands on every selected sheet.
Pairs naturally with 0103 Edit Sheet Parameter when an issue requires both a revision stamp and a parameter update across the same set.
0103 · REVIT SHEET

Edit Sheet Parameter

Compose a parameter value for many sheets from a concatenation rule instead of typing it sheet by sheet. Build the value from a prefix, a source parameter, and a suffix — optionally slicing the source with a substring.

  1. Select the sheets to update.
  2. Choose the target parameter — Built-in, Shared, or Project.
  3. Build the rule: Prefix + source parameter + Suffix.
  4. Optionally enable Substring with start/end index to slice the source value.
  5. Tick Include project information parameters to pull from project info.
  6. Stack rules with Add new parameter, then Apply.
0200

Revit View

View housekeeping at batch scale — duplication, naming, and link visibility without per-view surgery.

0201 · REVIT VIEW

Duplicate Revit View

Select any number of views and duplicate them in one operation — the setup step for coordination sets, markup copies, and discipline-specific variants.

  1. Search and select the source views.
  2. Choose duplication options.
  3. Run — duplicates are created for the full selection.
0202 · REVIT VIEW

Rename Revit View

Apply consistent naming across a batch of views in a single pass — no double-click-rename loops, no drifting conventions between floors.

  1. Select the views to rename.
  2. Define the renaming rule.
  3. Apply — names update across the whole selection.
0203 · REVIT VIEW

Edit Revit Links Visibility

Hide or show selected Revit links across any set of views — plans, sections, or 3D — and decide explicitly how view templates are treated instead of fighting them.

  1. Filter the view list by Plan, Section, or 3D.
  2. Search and select the target views.
  3. Select the Revit links to control.
  4. Choose template handling: Ignore views with templates or Override view templates.
  5. Pick Hide links or Unhide links, then Apply.
0300

Utilities

Model hygiene. Old templates, orphaned filters, abandoned working views — the weight a long project accumulates.

0301 · UTILITIES

Purge Unused

Four columns, four categories: view templates, view filters, views, and scope boxes. Search each, select what goes, and purge per category — surgical, not scorched-earth.

  1. Review each column — templates, filters (rule-based or selection), views, scope boxes.
  2. Use search to isolate candidates in large models.
  3. Select the unused elements in any column.
  4. Purge per category with its own button — each action is scoped and deliberate.
Purging is destructive by design. Run it on a synced, backed-up model and review selections before each purge — there is no undo across worksharing operations.
0302 · UTILITIES

Create Scope Box Selection Filter

Generate selection filters from the model's scope boxes, making zone-based selection a saved, reusable action instead of a manual ritual.

  1. Open the module — scope boxes are read from the model.
  2. Generate the selection filters.
  3. Reuse them anywhere Revit accepts a selection filter.
Built for zoned projects — towers split by scope box, phased slabs, or anything where "select everything in Zone 5" is a daily request.
